# zabbix监控交换机

## 1.交换机开启SNMP

在交换机中执行如下命令，设置一个只读的团体名，名称唯一

```shell
snmp-agent community read  public100
```



在zabbix-server中执行如下命令，会返回交换机中的所有配置信息

```shell
snmpwalk  -v 2c -c public100 10.0.0.123
```





## 2.zabbix添加主机

SNMP接口处填写交换机的IP，端口默认161
