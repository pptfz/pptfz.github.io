# shell变量自增

**a++是先执行表达式后再自增，执行表达式时使用的是a的原值**

**++a是先自增再执行表达示，执行表达式时使用的是自增后的a**



**a++示例**

```sh
$ a=1
$ let num=a++	#a++  先执行表达式，num=a，所以num的值为1，然后a再自增，值为2
$ echo $num
1
$ echo $a
2
```





**++a示例**

```sh
$ a=1
$ let num=++a	#a先自增，因此a的值为2，然后执行表达式，num=2，所以num的值为2
$ echo $num
2
$ echo $a
2
```
