## 布尔值

### 说明

:::tip 说明

布尔型数据在go语言中以 `bool` 类型进行声明，布尔型数据只有 `true(真)` 和 `false(假)` 两个值

:::



### 示例

代码示例

```go
package main

import "fmt"

func main() {
	var b1 bool = true
	var b2 bool = false

	fmt.Printf("布尔类型示例：\n")
	fmt.Printf("b1 && b2: %v\n", b1 && b2)
	fmt.Printf("b1 || b2: %v\n", b1 || b2)
	fmt.Printf("!b1: %v\n", !b1)
}
```

输出

```shell
布尔类型示例：
b1 && b2: false
b1 || b2: true
!b1: false
```
