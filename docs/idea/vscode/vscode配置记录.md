## 配置自动保存

[vscode配置自动保存](https://code.visualstudio.com/docs/editor/codebasics#_save-auto-save)

自动保存有以下3种策略

- `afterDelay` - 在配置的延迟（默认为 1000 毫秒）后保存文件

- `onFocusChange` - 当焦点移出文件的编辑器时保存文件

- `onWindowChange` - 当焦点移出 VS Code 窗口时保存文件
