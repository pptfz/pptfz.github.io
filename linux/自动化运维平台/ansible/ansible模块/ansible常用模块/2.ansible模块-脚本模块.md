# ansible模块-脚本模块

**ansible查看帮助方法**

```python
# 查看所有模块帮助信息
ansible-doc -l   

# 指定查看某个模块参数用法
ansible-doc copy  
```



## 1 script执行脚本模块

编辑测试脚本

```python
# 创建目录
[ -d /script ] || mkdir /script

# 编辑测试脚本
cat > /script/test.sh<<EOF
#!/usr/bin/env bash
yum -y install autoconf
EOF
```



在ansible本机运行，等同于在远程主机运行，不需要将脚本文件分别拷贝到目标主机执行

```shell
ansible all -m script -a "/script/test.sh"
```
