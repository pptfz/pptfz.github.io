# ansible模块-安装模块

**ansible查看帮助方法**

```python
# 查看所有模块帮助信息
ansible-doc -l   

# 指定查看某个模块参数用法
ansible-doc copy  
```



## 1 yum安装软件模块

```python
ansible all -m yum -a "name=httpd state=installed"
```



yum模块参数

| 参数    | 说明                                                         |
| ------- | ------------------------------------------------------------ |
| path    | 指定远程主机目录或文件信息                                   |
| recurse | 递归授权                                                     |
| latest  | 安装最新软件包                                               |
| state   | installed/present：安装软件包<br />removed/absent：移除软件包 |
